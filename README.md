# 📊 GA4 & GTM Playbook — Analítica Digital para Ecommerce

> Playbook de Analítica Digital: tracking plans, configuración avanzada de Google Tag Manager,
> guías de Google Analytics 4 y plantillas reutilizables orientadas a ecommerce.

## Sobre este repositorio

Este repositorio es mi base de conocimiento pública sobre analítica digital aplicada a **ecommerce**. Documento aquí metodologías, configuraciones y aprendizajes de mi trabajo diario con GA4 y GTM para que sirvan como referencia a otros profesionales y equipos.

**No es código. Es metodología.**

Cada guía incluye el *por qué* detrás de cada decisión, no solo el *cómo*. Porque configurar un evento es fácil; decidir *qué* medir y *para qué* es lo que marca la diferencia.
